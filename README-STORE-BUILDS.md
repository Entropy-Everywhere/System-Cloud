# System Cloud — Store Builds Guide

This directory contains everything needed to publish System Cloud to mobile and desktop app stores.

## 📁 What's Here

```
android/                       ← Google Play (Android TWA project)
  build.gradle                  - Project-level Gradle config
  settings.gradle               - Module settings
  gradle.properties             - Build properties
  gradle/wrapper/               - Gradle wrapper
  app/
    build.gradle                 - App module config
    src/main/
      AndroidManifest.xml        - TWA activity + intent filters
      res/values/strings.xml     - App name + launch URL
      res/values/themes.xml      - Transparent status/nav bars
      res/mipmap-*/ic_launcher.png  - App icons (all densities)
      res/mipmap-*/ic_launcher_round.png

.well-known/                   ← Digital Asset Links (deploy to your server)
  assetlinks.json               - Android app verification
  apple-app-site-association    - iOS universal links

windows/                       ← Microsoft Store
  package-msix.ps1              - MSIX packaging script
  AppxManifest.xml              - Generated by the script

stores/                        ← Store listings & assets
  google-play-listing.md        - Google Play description + instructions
  apple-app-store-listing.md    - App Store description + instructions
  microsoft-store-listing.md    - Microsoft Store description + instructions
  google-play-feature-graphic.png  - 1024×500 store banner

icons/                         ← App icons
  SystemCloudLogo.png           - Original 768×768 logo
  icon-192.png                  - PWA 192×192
  icon-512.png                  - PWA 512×512
  apple-touch-icon.png          - iOS 180×180
  icon-512.ico                  - Windows multi-res icon
```

## 🚀 Quick Start

### Prerequisites
- Your PWA deployed to a **public HTTPS URL** (required by all stores)
- A **developer account** for each store:
  - Google Play: $25 one-time
  - Apple App Store: $99/year
  - Microsoft Store: $19 one-time (or free with Visual Studio Dev Essentials)

### Google Play (Android)
1. Open `android/` in **Android Studio**
2. Edit `app/src/main/res/values/strings.xml` → set `launch_url` to your deployed PWA URL
3. **Build → Generate Signed Bundle / APK**
4. Upload the `.aab` to [Google Play Console](https://play.google.com/console)
5. Deploy `.well-known/assetlinks.json` to `https://your-domain.com/.well-known/assetlinks.json`

### Apple App Store (iOS)
1. Requires **macOS** with Xcode
2. Use [PWABuilder](https://pwabuilder.com) → enter your PWA URL → download iOS package
3. Or create a manual WKWebView wrapper (see `stores/apple-app-store-listing.md`)
4. Archive and upload via Xcode

### Microsoft Store (Windows)
1. Run `windows\package-msix.ps1` in a **Developer Command Prompt**
2. Sign the `.msix` with a code signing certificate
3. Upload to [Partner Center](https://partner.microsoft.com/dashboard)

### Itch.io (Desktop — already done)
- `dist/System-Cloud-Windows.zip` is ready to upload
- Built with Electron — runs as a standalone Windows app
